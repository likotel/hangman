## Hangman / Виселица

Классическая игра в угадывание слов. 
**Чтобы узнать об игре:**  [нажмите здесь.][1]

### Особенности работы данной программы:
1. Загадывается случайное слово из файла  ***words.txt*** *(hangman/data/words.txt*).
Чтобы добавить/удалить слово, достаточно открыть данный файл и внести нужные правки.
Новые слова стоит вписывать большими буквами с новой строки.

2. У игрока есть возможность допустить ошибку 6 раз, на 7 раз игра проиграна.
При угадывании букв принимается только первая, например:

| **Ввод пользователя** | **Что приняла программа** |
| --------------------- |:-------------------------:|
| я                     | я                         |
| яблоко                | я                         |

3. Буквы Е/Ё и И/Й взаимозаменяемы. 
Так что, например, при вводе буквы "Е" игра так же проверит слово на наличие буквы "Ё".

4. В этой игре присутствует псевдографика, с пощью которой отображается виселица. Состояние виселицы совпадает с количеством допущенных ошибок.
